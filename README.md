# Cuida Donde Pisas

Vistas correspondientes a este proyecto:

- `vistas/energia.html`
- `vistas/reciclaje.html`

El header y el footer estan separados en este repositorio se enceuntran en:

- `vistas/header.html`
- `vistas/footer.html`

## Carpetas

- `css/`: estilos del proyecto
- `assets/`: imagenes y recursos
- `vistas/`: paginas HTML

## Como abrir

Abrir los archivos HTML directamente en el navegador o usar un servidor local.
